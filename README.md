# Image & Video Compressor (MVP)

ブラウザ内で動作する画像・動画圧縮ツールです。  
WASMを使用し、サーバーへファイルを送信しません。

---

## 技術スタック

- React + Vite
- TypeScript
- browser-image-compression
- ffmpeg.wasm
- JSZip

---

## 必要環境

- Node.js 18以上推奨
- npm

---

## 初回セットアップ

```bash
# リポジトリ取得
git clone <your-repo-url>

cd image-compressor

# 依存関係インストール
npm install
